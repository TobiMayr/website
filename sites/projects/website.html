Website

This site is hosted on my Raspberry Pi using a LAMP-Stack and plain HTML, CSS and JavaScript

Software: HTML, CSS, JavaScript, ChartJS, Apache, Raspbian Stretch Lite, MySQL

In order to create this page I set up a server on my Raspberry Pi. The work is still in progress and needs many tweaks; especially with responsiveness. The design is inspired by the Zerif Lite WordPress theme.
