ZUCK - Zentrale Unterhaltungs- und Consumer Kontrolleinheit

A system that controls IoT devices using a server and many microcontrollers and sensors

Hardware: NanoPi NEO Air, Raspberry Pi Zero, ESP8266, Smart Bulbs and many sensors(humidity, temperature, infrared...)

Software: NodeJS, EJS, HTML, CSS, Ubuntu-Core

Part of my education at HTW-Berlin included an extensive group project worth 20 ECTS (Credit Points). The objective was to create an affordable home automation system with microcontrollers such as the ESP8266. Our Professor gave us a lot of freedom in the execution of the project and we decided to build a Smart Mirror which displays the data we receive from the sensors in a comprehensive overview. Smart Bulbs can be controlled via the same Interface on a tablet or smart phone.
My main focus point was the implementation of the Node JS server which processes the sensor data and dynamically generating the web interface.
The project obtained the highest mark of the semester (Summer 2017)!
